# A2CENSO API

##install dependences
-**npm install** 

##Insertar el archivo .env en la raiz del proyecto el cual debe contener las ##credenciales de la base de datos.
###ejemplo
    -**USER = 'userbd'**
    -**DATABASE_PASSWORD = '12345'**
    -**DATABASE_NAME = 'nameSchemabd'**
    -**DATABASE_HOST = '127.0.0.1'**
    -**DATABASE_PORT = 3307** 

##run proyect
 -option 1 :
    -**npm start**
 -option 2 :
    -**npm run dev**
